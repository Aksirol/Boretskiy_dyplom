# Система обліку апаратного забезпечення (Hardware AIS)

Ця програма є портативною і не потребує встановлення на комп'ютер.

## Як розпочати роботу:
1. Скопіюйте папку `HardwareAIS` на ваш комп'ютер (наприклад, на Робочий стіл або диск C:).
2. Зайдіть у папку і двічі клікніть на файл `HardwareAIS.exe`.
3. При першому запуску програма автоматично створить файл бази даних `hardware.db` поруч із собою. Усі ваші дані зберігатимуться саме в цьому файлі.

## Перенесення на інший комп'ютер / Резервне копіювання
Оскільки програма портативна, для створення резервної копії або перенесення на інший комп'ютер достатньо:
1. Закрити програму.
2. Скопіювати всю папку `HardwareAIS` на флешку (або скопіювати лише файл `hardware.db`, якщо ви хочете зберегти тільки дані).

## Важливо
- Не видаляйте файл `hardware.db`, інакше ви втратите всі свої дані (комп'ютери, приміщення, історію аудитів).
- Не запускайте програму безпосередньо з ZIP-архіву — спочатку розпакуйте папку.
